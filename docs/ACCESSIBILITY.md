# Accessibility Commitments

PrithviProof targets **WCAG 2.2 AA** compliance. We believe climate action tools should be accessible to everyone.

## Key Features
- **Keyboard Navigation**: All interactive elements are reachable via `Tab`.
- **Focus Management**: Modals and dialogs (like Ask Prithvi) utilize focus traps.
- **Skip Links**: A visually hidden "Skip to main content" link is available for screen readers and keyboard users.
- **Semantic HTML**: Proper use of landmarks (`main`, `nav`, `footer`) and hierarchical headings (`h1` -> `h6`).
- **Color Contrast**: The `forest`, `canvas`, and `surface` theme variables are designed to provide sufficient contrast ratios.

## Testing Accessibility
Accessibility is continuously tested manually and via automated tools (like `axe-core`).
